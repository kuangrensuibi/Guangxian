常见问题

免费试用与购买

App 提供 7 天免费试用，试用期内导出无水印。试用结束后导出会带水印，可一次性购买「专业版」永久去除水印，后续新功能免费升级。

换了手机，已购买如何恢复？

进入「会员中心」点击「恢复购买」，使用原 Apple ID 登录即可恢复永久权益，无需重复付费。

导出的视频有水印？

试用结束且未购买时导出会带水印。购买「专业版」后，导出即为无水印高清成片。

支持哪些设备？

支持运行 iOS 17.0 及以上系统的 iPhone（iPhone XS / XR 及更新机型）。

我的素材会被上传吗？

不会。所有调色、剪辑、导出均在本地设备完成，App 不联网上传任何照片或视频，隐私安全。

如何卸载 App？

在主屏幕长按「光显」图标，选择「移除 App → 删除 App」即可。已购买的专业版权益与 Apple ID 绑定，重新安装后在「会员中心」点「恢复购买」即可找回，不会丢失。

如何申请退款？

App 内购买由 Apple 统一处理，退款也需向 Apple 申请：前往 reportaproblem.apple.com，用购买时的 Apple ID 登录，找到「光显」对应的购买记录，选择「请求退款」并填写原因提交。也可在 iPhone「设置 → Apple 账户 → 媒体与购买项目 → 购买记录」中发起。是否退款由 Apple 审核决定。
